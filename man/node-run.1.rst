==============
node-run
==============

--------------------------------------------------------
Run system-wide Node.js
--------------------------------------------------------
:Version: node-run |version|
:Manual section: 1

Synopsis
========

node-run *[options]* *program* *[program_args]*

Description
===========

Force Node.js to run loading system-wide
installed libraries.

Options
========

-r retries_max          Maximum number of retries before
                        failing.

-h                      Display help.
-c                      Enable color output
-v                      Enable verbose output
